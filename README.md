# 📘 Sistema de Gestión Académica

Sistema web desarrollado en Django para la gestión de estudiantes, cursos y matrículas. Permite la administración completa mediante CRUD, autenticación de usuarios, API REST y panel administrativo.

---

## 🚀 Descripción del proyecto

Este sistema permite:

- Registro de estudiantes  
- Gestión de cursos  
- Matrículas académicas  
- Administración de usuarios  
- Consumo de API REST  
- Panel administrativo Django  

---

## 🛠️ Tecnologías utilizadas

- Python 3.x  
- Django  
- Django REST Framework  
- SQLite  
- HTML, CSS  
- Bootstrap  

---

## 📂 Estructura del proyecto

Aplicaciones principales:

- estudiantes  
- cursos  
- matriculas  
- api  
- usuarios  
- suficiencia_academica (configuración principal)
